== Description ==

Cheritto's Admin Posts Grid let you view the posts table as a grid of cards.

* several themes available;
* hide or show image, title, author, tags, categories and post date inside cards;
* adjustable number of columns;
* per-user preferences;
* quick edit support via ajax modal form.

== Installation ==

Just activate the plugin, you'll find a new admin menu page on the left: "Posts grid". Enjoy!
